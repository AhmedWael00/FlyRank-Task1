# CLAUDE.md - Project Guidelines

## Tech Stack & Architecture
- Frontend: React.js, Tailwind CSS, HTML5, CSS3, Bootstrap
- Styling: Tailwind CSS & Responsive design
- Build Tool: Vite
- AI Assistant Configuration: Tailored for Frontend Engineering and clean React state management

## Development Conventions
- Write clean, declarative React functional components.
- Use Tailwind CSS for responsive and modern UI layout patterns.
- Ensure all CSS classes are structured logically.
- Keep commits structured according to the Conventional Commits specification.
